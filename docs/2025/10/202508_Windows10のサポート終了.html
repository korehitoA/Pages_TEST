Windows 10 のサポート終了が迫っています
～ ご自宅でPCをご利用中の方へ ～

Microsoftが以前より告知していた通り、Windows 10 のサポートは 2025年10月14日 をもって終了します。
終了まで残り約2か月 となり、いよいよ対応のタイミングが差し迫ってきました。

Windows 10 を使っている方は、早めに Windows 11 へのアップグレード を検討されることをおすすめします。

1. サポート終了後に起こること

  • セキュリティ更新が受けられなくなる
    新たな脆弱性が見つかっても修正されず、ウイルス感染や不正アクセスのリスクが高まります。
  • ソフトやアプリが徐々に使えなくなる
    今後は Windows 11 以降を前提としたソフトやサービスが主流になります。
  • 無料アップグレードができなくなる可能性がある
    現在は無料で Windows 11 にアップグレードできますが、サポート終了後は有料化される可能性があります。

※法人向けには有償の延長サポート（ESU）が提供されますが、個人ユーザーは対象外です。
そのため、サポート終了後に Windows 10 を使い続けることは 完全に自己責任 となります。

2. アップデートしないことで影響を受ける主なサービス

  • Microsoft Teams / OneDrive / Office などのクラウドサービス
    バージョンの不一致によって連携エラーや利用制限が発生する可能性があります。
  • ウイルス対策ソフト（セキュリティソフト）
    対応OS外となると保護が不十分になる、もしくは更新されなくなる恐れがあります。
  • ネットバンキングや公的手続きサイト
    Windows 10が推奨環境から外れ、ログインや操作に制限がかかるケースがあります。
  • 動画配信・ゲームサービスなどのエンタメ系アプリ
    将来的なアップデートで非対応となる場合があります。

3. アップグレード前に確認しておきたいこと

  • 古いソフトウェアや周辺機器が動作しない可能性がある
    古いプリンター、スキャナー、DVD再生ソフトなどが対応していないことがあります。
  • カスタマイズや設定が初期化される場合がある
    テーマ、ショートカット、特殊な設定がリセットされることがあります。
  • ストレージや性能の不足でアップグレードできないことがある
    一定のスペックと空き容量が必要です。まずは事前確認をおすすめします。

データのバックアップと、利用中のアプリや機器の動作確認を必ず行ってください。

4. Windows 11 にアップグレードするには？

■ 必要なシステム要件（概要）

  • 64bit CPU
  • TPM 2.0 搭載
  • UEFI BIOS（レガシーBIOS非対応）
  • メモリ 4GB以上
  • ストレージ 64GB以上

■ アップグレード方法

  • Windows Update に案内が表示されていれば、そこからアップグレード可能
  • 表示されない場合は、「Windows 11 インストールアシスタント」を使用

■ 対応確認方法

以下のURLから「PC 正常性チェック アプリ」をダウンロードできます。
https://www.microsoft.com/ja-jp/windows/windows-11#pchealthcheck
